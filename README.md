# Data Structures

This repository contains C# implementations of the following data structures, with accompanying unit tests:

- Stack (Array Stack)
- Queue (Array Queue)
- Binary Tree
- Binary Search Tree
- Circular Buffer
- Linked List

## Requirements

- .NET Core 2.2
- .NET CLI

## Setup

After cloning the repository, run

```sh
dotnet restore --packages ./packages
```

## Testing

To run tests execute

```sh
dotnet test
```
